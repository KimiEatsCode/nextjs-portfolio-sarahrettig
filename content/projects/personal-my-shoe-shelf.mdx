## Key Deliverables

- Full-stack social app for shoe lovers — photo posts, home feed, follows, and public profiles
- Customizable profiles with accent theme presets (Classic, Bubblegum, Emo Night, Skate Park, Chrome)
- Top Shelf feature — curate a featured row of favorite shoes or collectors on your profile
- Auth with sign up, login, and secure sessions; beta invite flow with email delivery via Resend
- Image uploads with validation; Vercel Blob in production and local storage in development
- Deployed on Vercel with PostgreSQL (Prisma) for users, posts, follows, and moderation

## Tools & Technologies

- Next.js (App Router) and React
- TypeScript and Tailwind CSS v4
- Prisma and PostgreSQL
- Auth.js (credentials + JWT sessions)
- Vercel Blob for image storage
- Resend for transactional email
- Zod for validation
- bcryptjs for password hashing
